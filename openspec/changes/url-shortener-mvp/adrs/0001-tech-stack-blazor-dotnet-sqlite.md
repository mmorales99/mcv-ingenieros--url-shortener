# ADR 0001: Use Blazor + ASP.NET Core + SQLite

## Status
Accepted

## Context
The project needs a cohesive .NET-first stack with rapid setup for MVP delivery and manageable operational complexity.

## Decision
- Frontend: Blazor
- Backend: ASP.NET Core (.NET)
- Database: SQLite (MVP baseline)

## Consequences
- Strong .NET ecosystem cohesion.
- Fast local development and deployment simplicity.
- Possible future migration path to PostgreSQL if scale demands it.
